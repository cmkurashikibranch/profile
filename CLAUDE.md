## 編集ポイント（index.html 内）

`<!-- ▼ 書き換え -->` コメントを目印に編集する。

| 箇所 | 内容 |
|------|------|
| `<title>` / `<meta description>` | タブ名・SNS表示テキスト |
| `.profile` 内 | 名前・肩書き・キャッチコピー |
| `.avatar` | 文字アイコン → `<img src="photo.jpg">` に差し替え可 |
| `.cards` 内 `.card` | スキル項目（コピペで増減可） |
| アクセント色 | `:root` の `--accent` を変更 |
